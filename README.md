# Acme Demo

Personalized Acme landing page demo for Revolut, built with Next.js App Router, React, Tailwind CSS, and a custom Three.js globe.

## Local Development

```bash
pnpm install
pnpm dev
```

Open `http://127.0.0.1:3000/3d`.

## Verification

```bash
pnpm lint
pnpm test -- --run
pnpm build
```

The app deploys to Vercel as `acme-demo`.
